# Definition (Positive Definite)
A matrix $M \in \R^{n \times n}$ is called **positive definite** iff $\forall \vec{x}\in \R^{n}, \vec{x}^{T}M\vec{x} > 0$. We denote this property by $M \succ 0$. 

# Definition (Positive Semi-definite)
A matrix $M \in \R^{n \times n}$ is called **positive definite** iff $\forall \vec{x}\in \R^{n}, \vec{x}^{T}M\vec{x} \geq 0$. We denote this property as $M \succeq 0$.
